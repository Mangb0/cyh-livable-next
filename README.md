# cyh-livable-next

**리버블 프론트엔드 과제**

- a 페이지

  - 글자영역 CSR 구현

  - /api/hello 를 통한 data fetch

- b 페이지

  - 글자영역 SSR 구현

  - utils의 getTimeoutData 함수(data fetch 3초 후 return) 호출

- c 페이지

  - 페이지 전체 CSR 구현
  - /api/hello 를 통한 data fetch

- Title 컴포넌트

  동일한 UI로 인한

## 개발환경 실행

의존성 설치

```bash
npm install
```

실행

```bash
npm run dev
```
